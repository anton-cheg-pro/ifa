# IFA — Independent Financial Advisor

Навчальний веб-сайт **незалежного фінансового радника**: освітні матеріали та інструменти для особистих фінансів і інвестицій. Це навчальний проєкт — не комерційний продукт і не персональна інвестиційна рекомендація.

## Мета проєкту

Допомогти користувачам **розуміти** власні фінанси та приймати рішення самостійно. Сайт поєднує:

- зрозумілий контент українською;
- калькулятори та інструменти (заплановано);
- чіткі дисклеймери та пояснення ризиків.

**Позиціонування:** освіта + інструменти, без обіцянок доходності. Користувач сам відповідає за інвестиційні рішення.

## Для кого

- люди, які хочуть навчитися планувати бюджет і заощадження;
- ті, хто цікавиться інвестиціями на базовому рівні;
- користувачі, яким потрібні **загальні** пояснення, а не індивідуальна порада.

## Заплановані теми (домен)

| Тема | Приклади |
|------|----------|
| Бюджет | доходи/витрати, резервний фонд (3–6 місяців витрат) |
| Заощадження | депозити, інфляція, реальна дохідність |
| Інвестиції | акції, облігації, ETF, диверсифікація, горизонт, ризик |
| Пенсія | пенсійні стовпи (Україна), довгострокове накопичення |
| Страхування | життя, здоров'я — за потребою |
| Податки | лише загальний рівень; за подачу — до фахівця |

Кожна фіча з цифрами проходить специфікацію від `finance-analyst` перед реалізацією.

## Дисклеймер

> Інформація на сайті має освітній характер і не є індивідуальною інвестиційною рекомендацією. Рішення про інвестування приймаєте ви. Минула дохідність не гарантує майбутніх результатів. Зверніться до ліцензованого радника або юриста за персональною консультацією.
